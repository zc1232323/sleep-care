# SleepCare 使用指南（小白版）

> 不用看代码，跟着步骤双击就行 ✅

---

## 📦 第一步：安装（只需做一次）

双击文件夹里的：

```
一键安装.bat
```

等黑色窗口显示「安装完成」就行了（1-2 分钟）。这一步只需要做**一次**。

---

## 🚀 第二步：启动后端（每次用都要做）

双击：

```
启动后端.bat
```

看到黑色窗口里出现「SleepCare 后端服务已启动」就对了。**这个窗口不要关！**

---

## 📱 第三步：打开小程序

1. 打开**微信开发者工具**（桌面上的那个）
2. 点「导入」→ 选择这个 `sleep-care` 文件夹
3. AppID 选「测试号」
4. 点「确定」→ 编译

小程序就跑起来了。

---

## 🧪 可选：用 Postman 测接口

1. 打开 Postman → 左上角 Import
2. 选 `SleepCare_API.postman_collection.json`
3. 再选 `SleepCare_环境.postman_environment.json`
4. 先点「登录」→ Send
5. 再点其他接口随便测

---

## ❓ 遇到问题

| 现象 | 怎么办 |
|------|--------|
| 双击 bat 闪退 | 鼠标右键 → 以管理员身份运行 |
| 提示「没有 Node.js」 | 去 https://nodejs.org 下载安装 |
| 小程序白屏 | 确认「启动后端.bat」还开着 |
| Postman 报 401 | 先跑「登录」接口，再跑其他的 |
