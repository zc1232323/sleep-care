# MVP 功能清单

> 智能睡眠健康管理软件（SleepCare）— 最小可行产品功能清单

---

## 模块一：用户管理

| 功能 | 优先级 | 描述 |
|------|--------|------|
| 用户注册 | P0 | 手机号 + 验证码注册 |
| 用户登录 | P0 | 手机号 + 密码 / 验证码登录 |
| 个人信息管理 | P1 | 头像、昵称、性别、出生日期、身高、体重 |
| 修改密码 | P1 | 旧密码校验 → 设置新密码 |
| 第三方登录 | P2 | 微信授权登录 |

## 模块二：设备管理

| 功能 | 优先级 | 描述 |
|------|--------|------|
| 设备绑定 | P0 | 通过蓝牙或扫码绑定智能穿戴设备 |
| 设备解绑 | P1 | 解除已绑定设备 |
| 设备列表 | P0 | 查看已绑定设备列表及在线状态 |
| 设备数据同步 | P0 | 从设备同步心率、血氧、体动等原始数据 |

## 模块三：睡眠报告

| 功能 | 优先级 | 描述 |
|------|--------|------|
| 每日睡眠报告 | P0 | 入睡时间、起床时间、深睡/浅睡时长、睡眠评分 |
| 周度睡眠趋势 | P1 | 本周睡眠数据汇总与趋势图表 |
| 月度睡眠趋势 | P2 | 本月睡眠数据汇总与趋势图表 |
| 睡眠建议推送 | P1 | 基于数据的个性化睡眠改善建议 |
| 报告分享 | P2 | 一键生成睡眠报告图片并分享 |

## 模块四：用户设置

| 功能 | 优先级 | 描述 |
|------|--------|------|
| 睡眠目标设置 | P1 | 设定目标入睡时间、起床时间、目标睡眠时长 |
| 提醒设置 | P1 | 入睡提醒、起床提醒开关及时间 |
| 隐私设置 | P1 | 数据分享权限、报告可见范围 |
| 通知设置 | P2 | 推送通知开关及免打扰时段 |

## 模块五：医生授权

| 功能 | 优先级 | 描述 |
|------|--------|------|
| 授权医生查看 | P1 | 用户授权指定医生查看自己的睡眠数据 |
| 取消授权 | P1 | 用户取消对医生的数据查看授权 |
| 授权列表 | P1 | 查看当前已授权的医生列表 |
| 医生端数据查看 | P2 | 医生查看已授权用户的睡眠报告 |

---

### 优先级说明

| 等级 | 含义 | 交付时间 |
|------|------|----------|
| P0 | 核心功能，MVP 必须实现 | 第一版 |
| P1 | 重要功能，MVP 迭代实现 | 第二版 |
| P2 | 增值功能，后续迭代 | 第三版+ |
