## 三、核心模块交互流程

### 3.1 用户注册/登录流程

```
用户 ──► 前端 H5 ──► Nginx ──► User Service ──► MySQL
  │                                                    │
  └──── JWT Token ◄──────────── 鉴权成功 ◄──────────────┘
```

### 3.2 设备数据同步流程

```
智能设备 ──► 蓝牙 ──► 前端 H5 ──► Device Service ──► MySQL (原始数据)
                                            │
                                            ▼
                                      Report Service ──► MySQL (分析报告)
```

### 3.3 医生授权查看流程

```
医生 ──► 请求查看 ──► Auth Service ──► 校验授权关系 ──► MySQL
                                                    │
                    医生 ◄── 返回报告 ◄── 有权限 ◄─────┘
```

## 四、技术选型理由

| 技术 | 理由 |
|------|------|
| Vue 3 | 轻量、生态丰富、适合 H5 移动端开发 |
| Vant UI | 专注移动端的 Vue 组件库，开箱即用 |
| TypeScript | 类型安全，减少运行时错误 |
| Node.js | 前后端统一语言，开发效率高 |
| MySQL 8.0 | 成熟稳定，支持 JSON 字段，适合结构化数据 |
| Redis | 高性能缓存，支持 Session 管理和实时数据 |
| Docker | 环境一致性，简化部署流程 |
| Nginx | 高性能反向代理，支持负载均衡和 HTTPS |
