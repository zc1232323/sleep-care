## 截图③a：Postman — 添加医生授权

### 操作

1. 重启后端：`cd backend && node app.js`（因为新增了 `/api/doctor` 路由！）
2. 先登录获取 token：
   - POST `http://127.0.0.1:3000/api/v1/auth/login`
   - Body: `{"phone":"13800138000","password":"123456"}`
   - 复制 token
3. 新建请求：
   - Method: **POST**
   - URL: `http://127.0.0.1:3000/api/doctor/grant`
   - Headers:
     - `Authorization: Bearer <token>`
     - `Content-Type: application/json`
   - Body → raw → JSON:
     ```json
     { "phone": "13811111111" }
     ```
4. 点 Send

### 期望响应

```json
{
  "code": 0,
  "message": "授权请求已发送",
  "data": {
    "doctor_name": "李医生",
    "doctor_phone": "13811111111",
    "status": "pending",
    "expire_date": "2026-07-07"
  }
}
```

**验证要点**：code=0，status="pending"，expire_date 是 7 天后 ✅

## 截图③b：Postman — 查看已授权列表

### 操作

1. Method 改为：**GET**
2. URL: `http://127.0.0.1:3000/api/doctor/granted`

4. 点 Send

### 期望响应

## 常见问题

| 问题 | 解决 |
|------|------|
| POST grant 返回 1002 "未找到该医生" | 先运行 `node create-test-doctor.js` |
| 返回 401 | 重启后端，重新登录拿新 token |
| 小程序页面白屏 | 编译后再次点击按钮 |
| 添加按钮无反应 | 检查输入框是否是 11 位纯数字 |
