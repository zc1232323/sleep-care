## 四、Postman 具体操作（必须截图！）

第13大节要求用 Postman 测试线上注册/登录。下面是每一步的具体操作。

### 步骤 A：打开 Postman

1. 启动 Postman（如果没有，从 `tools.rar` 里找，或官网下载）。
2. 点击「New」→「HTTP Request」。
3. 新建一个 Collection（集合），命名为 `SleepCare-线上验证`。

---

### 请求 1：测试注册（POST）

**请求配置：**
- **Method**：`POST`
- **URL**：`https://xxx.app.tcloudbase.com/api/v1/auth/register`
  - 把 `xxx` 换成你的真实公网地址前缀
- **Headers**：
  - `Content-Type`：`application/json`
- **Body**（raw / JSON）：

```json
{
  "phone": "13900139000",
  "password": "123456",
  "nickname": "线上测试用户",
  "role": "patient"
}
```

**必须截图的内容：**
1. 请求 URL 是公网地址
2. Body 中的 JSON 参数
3. 响应状态 `200 OK`
4. 响应 Body 中 `code: 0`，`data` 包含 `phone` 和 `token`

**示例响应：**

```json
{
  "code": 0,
  "message": "注册成功",
  "data": {
    "id": 1,
    "phone": "13900139000",
    "nickname": "线上测试用户",
    "role": "patient",
    "token": "eyJhbGciOiJIUzI1NiIs..."
  }
}
```
