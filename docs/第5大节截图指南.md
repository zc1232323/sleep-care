## 📸 截图 3：Postman 分期接口测试 ①（第一次请求）

### Postman 操作：

| 项目 | 填写内容 |
|------|---------|
| **方法** | `GET` |
| **URL** | `http://localhost:3000/api/sleep/report/stages?date=2026-06-28` |
| **Headers → Authorization** | `Bearer eyJhbGci...(粘贴完整token)` |

### 截图应包含（Response Body）：

```json
{
  "code": 0,
  "message": "success",
  "data": {
    "stages": [
      { "time": "22:30", "stage": 1 },
      { "time": "22:35", "stage": 0 },
      { "time": "22:40", "stage": 0 },
      ... (共48个数据点)
      { "time": "06:15", "stage": 0 },
      { "time": "06:20", "stage": 0 },
      { "time": "06:25", "stage": 0 }
    ],
    "count": 48,
    "source": "generated"  // 或 "db"/"converted"
  }
}
```

**重点**：
- ✅ `count: 48` （必须是48个数据点！）
- ✅ `stages` 数组包含 `{time, stage}` 对象
- ✅ `stage` 值为 0/1/2/3 的数字
